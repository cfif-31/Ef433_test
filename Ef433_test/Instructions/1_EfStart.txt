1.Создать проект .Net Framework
2.Создадим в проекте папку Classes
3.Добавить в папку новый элемент -> Data -> ADO_.Net_EntityDataModel
4.Выбрать пустую модель CodeFirst.
5.Прописать над классом аннотацию (в []) DbConfigurationType, передав в аргументах тип (typeOf) MySQlEfConfiguration.
6.В удобном месте сгенерировать строку подключения к БД.
7.Скопировать сгенерированную стркоу, вставить ее в аргументы родительского конструктора base(/*строка*/)
8.Создадим классы сущностей:
	1.В папке Classes создадим папку сущностей (Entityes).
	2.Создать в папке Entityes класс, сделав его публичным. Пропишем в нем все поля будущей таблицы.
	//public int id {get;set;}
	3.Пропишем атрибуты полей:
		1. Key - первичный ключ (автоинкремент)
		2. Required - обязательное поле.
		3. MaxLength - максимальеный размер строк/байтов
		4. Index(IsUnique = true) - уникальность значения
9.В классе модели добавить запись о таблице, прописав ее как DbSet</*тип*/>
//public virtual DbSet<User> Users{get; set;}
10. Сделаем миграцию:
		1. Открыть консоль Tools->Nuget->Nuget Console
		2. Включить миграцИИ - enable-migrations
		3. Если миграция успешно включилась - откроется файл конфигурации (Migrations\Configuration)
		4. В файле удобно включить автомиграции (AutomationsMigrationsEnabled = true)
		5.Обновить базу данных на основе нашей модели - update-database
